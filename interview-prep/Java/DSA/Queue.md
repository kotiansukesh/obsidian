* Can be implemented with either an [[Array]] or a [[Linked List]]
* Abstract data type with the following operations
  
	|Operation|Meaning||
	|-|-|-|
	|Enqueue(Key)|adds key to collection|Use ```List.PushBack```|
	|Key Dequeue(Key)|Removes and returns least recently-added key|Use ```List.TopFront``` and ```List.PopFront```|
	|Boolean Empty()|Are there any elements?|Use ```List.Empty```

* Often called FIFO(First In First Out) distinguishes them from Stack's LIFO
